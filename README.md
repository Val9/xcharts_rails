# xcharts_rails

A gem to include Xcharts into Rails 3.1+ Assets Pipeline

## Installation

Add this line to your application's Gemfile:

    gem 'xcharts_rails'

And then execute:

    $ bundle install

After that execute:

    $ rails g xcharts:install

This should add `xcharts` to `application.js` and `application.css`
